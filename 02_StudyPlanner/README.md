# Study Planner (스터디 플래너)
JavaScript 기반 학습 일정 등록 및 관리 앱이다.
학습 계획을 등록하고 마감일 기준 D-Day확인, 진행 상태 관리를 할 수 있으며
마감일 순 정렬 기능을 통해 학습 우선순위를 확인할 수 있도록 구현하였다.
<br>

## Tech Stack
- HTML
- CSS
- JavaScript
<br>

## 주요 기능

### 계획 등록
- 과목명, 상세 내용, 마감일 입력
- 입력 데이터 객체 형태로 관리

### D-Day 계산
- Date 객체를 활용한 날짜 계산
- 마감일까지 남은 기간 표시

### 완료 상태 관리
- 체크박스를 이용한 완료 여부 변경
- 완료된 계획 스타일 변경

### 계획 삭제
- 배열 데이터 삭제
- 화면 재렌더링 처리

### 마감일 정렬
- sort() 메서드를 활용한 날짜 기준 정렬
- 마감 임박 순 출력
